# Modes

AURA AI has three modes. Switch between them with voice or the UI toggle.

---

## Assistant Mode (Default)

AURA responds conversationally. Full sentences, helpful explanations, natural back-and-forth.

Good for: questions, casual use, creative tasks.

```
You: "What's the capital of Japan?"
AURA: "The capital of Japan is Tokyo — it's also the most populous city in the world."
```

---

## Agent Mode

AURA takes autonomous multi-step actions. It plans, executes, and reports back — without asking for confirmation at every step.

Good for: complex tasks, automation chains, file operations.

```
You: "Download the top 5 Python repos from GitHub trending and create a summary PDF"
AURA: [searches → opens browser → scrapes → creates PDF → confirms done]
```

---

## Focus Mode

Minimal output. AURA only speaks when it needs to confirm something or when a task is done. No small talk.

Good for: work sessions where you just want things done quietly.

```
You: "Play lo-fi music on YouTube"
AURA: "Done." [plays music]
```

---

## Switching Modes

**By voice:**
```
"Switch to Agent mode"
"Go to Focus mode"
"Back to Assistant mode"
```

**By UI:** Click the mode toggle in the top-right of the AURA interface.
