# Contributing to AURA AI

Thanks for your interest! AURA is a solo-built project — contributions, bug reports, and ideas are all welcome.

---

## Getting Started

```bash
git clone https://github.com/HariKumar-DS/aura-ai.git
cd aura-ai
pip install -r requirements.txt
npm install
cp .env.example .env
# Add your GEMINI_API_KEY
python main.py
```

---

## How to Contribute

**Bug Reports** — Open an issue with steps to reproduce, your OS, and Python version.

**Feature Requests** — Open an issue describing what you want and why.

**Pull Requests** — Fork the repo, make your changes, and open a PR with a clear description.

---

## Guidelines

- Keep PRs focused — one feature or fix per PR
- Test your changes before submitting
- Write clear commit messages
- Don't break existing voice commands

---

## Areas That Need Help

- macOS and Linux compatibility
- More automation integrations (Telegram, Slack, etc.)
- Better error handling for edge cases
- UI improvements
- Documentation

---

If AURA helped you, a ⭐ on the repo is the easiest way to say thanks.
